## О проекте

Исследование рынка труда Data Science за 2020-2022 годы. В проекте исследованы различные связи и закономерности. Для анализа данных применялись популярные python-библиотеки.  
Цель проекта: показать навыки владения библиотеками python для анализа данных, а также понимание статистических тестов и их применимости к различным ситуациям.  

## Запуск проекта

Создайте виртуальное окружение. Например, если вы на Linux это можно сделать командой: `python -m venv .venv `  
Далее выполните: `pip install -r requirements.txt`
